Detailed Description

Gère la logique de la phase de jeu (Votes, Révélation, Calculs, Règles Spéciales).

@attributes model Modèle de session de jeu. main_controller Contrôleur principal pour la navigation. deck Liste des valeurs de cartes disponibles. current_player_index Index du joueur courant. revealed Indique si les votes sont révélés.
